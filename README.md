
# 项目名称：基于微信小程序的饭点高峰期解决方案——“饭堂帮”

## 项目简介
问题：

1、高校和单位餐厅在就餐高峰期过于拥堵，导致效率低下的现象。

2、餐厅由于供大于求而出现大量食材浪费的现象。

3、餐厅大多没有外卖的现象。

方案：

采用提前订餐，后台推送，号码牌机制，现场仅刷卡，提供外卖等方式缓解这些情况。
